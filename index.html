SMART STUDY MONITORING

The most important part of the Smart Study Monitoring project is its database. The database is full of books, exercises, practical and notes. Smart Study Monitoring is a web-based project where a user may search and access data for improving their skills. The system allows the user to check various courses as per there need. The system checks for user choice and then loads all that in front of the user. When the user courses the system allows the user to access various gradational courses. Thus, this system automates the working of managing study material in standard manner and allows users to access them online through the website. One more important functionality available in this project if needed is that user can get more information from the search bar with smart AI filter.
